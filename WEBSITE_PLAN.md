- [ ] **Color Palette:** Implement a dark neon theme:
  - Background: Deep Navy/Black (`#050810`)
  - Primary Glow: Cyan (`#00f2ff`)
  - Accent Glow: Purple (`#b222f2`)

- [ ] **Typography:** Use modern, bold sans-serif fonts (Inter/Orbitron) for that "tech" feel.
- [ ] **Interactive Buttons:** "View Source Code" button with high-contrast hover states and glows.

## Phase 3: Features & Polish

- [ ] **System Requirements Footer:** A clean, minimal footer showing RAM, Model, and Vision requirements as seen in the ref.
- [ ] **Responsive Layout:** Ensure the "gamer vibe" translates perfectly to mobile and desktop.
